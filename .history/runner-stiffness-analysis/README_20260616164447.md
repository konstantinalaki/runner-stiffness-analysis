# Runner Stiffness Analysis

This project analyzes runner measurements and assigns a stiffness profile (Soft, Medium, Firm) using a small data pipeline implemented in Python.

**Key artifacts**
- `data/raw/runners.csv` — raw dataset
- `data/processed/runners_cleaned.csv` — cleaned dataset produced by the processing pipeline
- `data/clustered/runners_clustered.csv` — cleaned dataset with assigned stiffness profiles

**Code**
- `src/data_processing.py` — loading, cleaning, and outlier removal; exposes `main()` which writes `data/processed/runners_cleaned.csv`
- `src/clustering.py` — computes a `load_score` and assigns `stiffness_profile`; exposes `main()` which writes `data/clustered/runners_clustered.csv`

## Quickstart

1) Install dependencies (recommended to run from the repository root):

```bash
python -m pip install pandas ruamel.yaml
```

2) Run the pipeline scripts directly from the repository root:

```bash
python src/data_processing.py
python src/clustering.py
```

## Configuration

The repository uses `config.yaml` (root) to store thresholds and clustering parameters used by `src/data_processing.py` and `src/clustering.py`.

## Notes

- Output files are written to `data/processed/runners_cleaned.csv` and `data/clustered/runners_clustered.csv`.
